# Personal Academic Website

This repository contains the source code for my personal academic website, where I share information about my background, research experience, current projects, and writing.

---

## Built With

- [Quarto](https://quarto.org/) for website generation  
- Git and GitHub for version control  
- GitHub Pages for deployment  
- Bootstrap theme with custom CSS styling  

---

## Project Structure

- `.qmd` files contain the source content for each page  
- `_quarto.yml` defines the website configuration  
- `styles.css` contains custom styling  
- The `docs/` directory contains the rendered HTML files  

---

## Deployment

The website is rendered locally using:

```bash
quarto render
